# AI Planner

업무 일정을 관리하기 위한 **풀페이지 크롬 확장프로그램**입니다.  
달력 중심으로 일정을 보고, 프로젝트/종류별로 관리하며, AI와 대화해서 일정 추가/수정/삭제 제안까지 받을 수 있습니다.

## 한눈에 보기
- 대시보드: 월간 달력 + 날짜별 일정 + 주요 일정 + 전체 메모
- 일정 관리: 정렬 중심 목록, 카드 클릭으로 수정, 상태 빠른 변경
- 프로젝트 관리: 프로젝트별 일정 모아보기 + 프로젝트 설명 관리
- 지난 업무: 완료 + 지난 일정 모아보기(필터 지원)
- 설정: 시간 형식/주 시작 요일, 백업 내보내기/가져오기, LLM 설정
- AI 도우미: 자연어 요청 → 변경안 제시 → 최종 확인 후 반영

## 기본 동작 방식
- 데이터는 브라우저 로컬(IndexedDB)에 저장됩니다.
- 일반 일정 기능은 인터넷 없이 동작합니다.
- AI 도우미는 설정된 LLM 서버에 연결되어야 동작합니다.

## 설치 방법(사용자)
빌드된 `dist` 폴더가 있을 때:
1. 크롬에서 `chrome://extensions` 열기
2. 우측 상단 `개발자 모드` 켜기
3. `압축해제된 확장 프로그램을 로드합니다` 클릭
4. `dist` 폴더 선택

## 개발/빌드 방법(개발자)
```bash
npm install
npm run lint
npm run build
```

빌드가 완료되면 확장 프로그램 배포 산출물은 `dist` 폴더에 생성됩니다.

## AI 도우미 설정
1. 앱의 `설정` 페이지로 이동
2. `LLM 모델명`, `LLM API Key` 입력
3. `AI 도우미` 페이지에서 자연어로 요청

AI 도우미는 다음 순서로 동작합니다.
1. 사용자 요청 해석
2. 필요한 경우 일정/프로젝트/종류 조회
3. 추가/수정/삭제 변경안 생성
4. 사용자 최종 확인 후 실제 반영

LLM API Endpoint는 코드에 고정되어 있습니다.
- `src/constants.ts`의 `LLM_CHAT_COMPLETIONS_URL`
- 기본값: `http://127.0.0.1:3000/api/chat/completions`

## 내부망/오프라인 사용
- 일반 일정 관리 기능: 오프라인 사용 가능
- AI 도우미 기능: 내부망에서 접근 가능한 LLM 서버가 필요

## 백업/복원
- 설정 페이지에서 JSON으로 내보내기/가져오기 지원
- 다른 환경으로 데이터 이전할 때 사용 가능
