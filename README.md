# Work Schedule Manager Chrome Extension

Full-page Chrome extension for managing work schedules with calendar, project/type classification, and archive behavior for past-completed tasks.

## Core Pages
- Dashboard (`#/dashboard`)
  - Monthly calendar
  - Daily task summary
  - Major task section
  - Date memo section
- Tasks (`#/tasks`)
  - Create/update/delete tasks
  - Status management (`미완료 / 보류 / 완료`)
  - Search and filters
- Projects (`#/projects`)
  - Project color/name/description/active management
- Types (`#/types`)
  - Default types + custom types management
- Archive (`#/archive`)
  - Completed and past tasks only
  - Restore by changing status
- Settings (`#/settings`)
  - Week start day, time format
  - JSON export/import

## Data Model Summary
- Task: title, content, type, project, status, start/end, major flag
- Task metadata: `createdAt`, `updatedAt`, `completedAt`
- Project: name, color, description, active flag
- Type: name, color, default/custom, active flag, order
- Memo: per date
- Setting: show past-completed toggle, calendar/time preferences

## Past Task Visibility Rule
- Default: hide tasks that are both `DONE` and already past.
- Top toggle: `지난 완료 업무 보기` turns hidden past-completed tasks on.
- Archive page: always shows completed + past tasks.

## Build / Run
```bash
npm install
npm run lint
npm run build
```

## Load Extension in Chrome
1. Open `chrome://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select the generated `dist` folder

## Notes
- Build is generated by `scripts/build-extension.mjs` (esbuild-based) to produce a stable extension bundle.
- MV3 files are in `public/manifest.json` and `public/background.js`.
- Full architecture document: `docs/EXTENSION_ARCHITECTURE.md`
